# Initialise New Git Project

## Usage

```
./init-project.sh [PROJECTNAME] [OPTIONS]...
```

## Optional flags

```
-l, --licence [LICENCE]
  Create licence (MIT, Apache2, GPL2, GPL3)
  
--dir [DIRECTORY]
  Where to create your new project, default: "./"
  
--no-git
  Do not initialise as git repository
  
```
